# Solving Linear Programming Problems using Simplex Algorithm

This project was developed as part of the Operations Research course and implements the Primary Simplex Algorithm for solving linear programming problems.

The application allows users to input constraints and an objective function, then computes the optimal solution using the Simplex method.

To enhance usability, the project includes a graphical user interface built with CustomTkinter, designed to match the theme of the presentation.


## Team

- [@imj31us4am50](https://github.com/imj31us4am50) - GUI design & functionality
- [@adypicolo](https://github.com/adypicolo) - Simplex algorithm & integration  


## Features

- Simplex-based problem solver  
- Step-by-step table visualization  
- Optimal solution computation  
- CustomTkinter GUI
